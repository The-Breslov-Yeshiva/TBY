# Security Policy

## Reporting

Do not open a public GitHub issue for security concerns.

Report suspected leaks, exposed credentials, payment misuse, or privacy concerns to:

`info@tbye.org`

## Secrets

This repository must not contain private keys, API keys, access tokens, service account credentials, `.env` files, Cloudflare local state, or deployment keys.

Deployment credentials belong only in GitHub Actions secrets or local ignored files.

## Public Content

Images, logos, copy, donation links, and form endpoints in this repository should be treated as public website content. Do not commit private student records, donor records, internal notes, or non-consented photos.
